# 🏠 HomeSense v0.1

A smart home monitoring mobile application UI/UX prototype designed in Figma.

## 📖 Overview

HomeSense is a modern smart home application concept that allows users to monitor and control different areas of a home from a clean and intuitive interface.

This project is the first UI prototype (v0.1) created in Figma as part of my UI/UX learning journey.

---

## ✨ Features

- Splash Screen
- Home Dashboard
- Devices Screen
- Lights Screen
- Room Details
  - Living Room
  - Kitchen
  - Bedroom
  - Bathroom
- User Profile
- Bottom Navigation
- Interactive Prototype Navigation

---

## 🔄 Prototype Flow

Splash Screen
